## ✨ المميزات الرئيسية

### 🤖 الذكاء الاصطناعي المتقدم
- **توليد تلقائي للخرائط الذهنية** من النصوص والملفات
- **توسيع ذكي للعقد** باستخدام Google Gemini 2.5
- **تحليل المشاعر** للأفكار والمحتوى
- **اقتراحات ذكية** مبنية على السياق

### 🎨 واجهة مستخدم حديثة
- **تصميم عصري وبديهي** مع دعم الوضع الليلي
- **سحب وإفلات تفاعلي** لتنظيم العقد
- **رسوم متحركة سلسة** وتأثيرات بصرية جذابة
- **دعم كامل للغة العربية** مع التخطيط من اليمين لليسار

### 🔧 أدوات متقدمة
- **تصدير متعدد الصيغ**: JSON, PNG, PDF, SVG
- **استيراد ومشاركة** الخرائط الذهنية
- **حفظ تلقائي** للبيانات
- **تخصيص الألوان والتصميم**

### 📱 تجربة متكاملة
- **متوافق مع جميع الأجهزة** (كمبيوتر، لوحي، هاتف)
- **أداء سريع** مع تقنية React و Vite
- **لا حاجة للتثبيت** - يعمل مباشرة في المتصفح

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/your-username/mindforge.git
cd mindforge

# تثبيت المكتبات
npm install

# تشغيل التطبيق في وضع التطوير
npm run dev
```

### النشر على Netlify

```bash
# بناء المشروع للإنتاج
npm run build

# معاينة البناء محلياً
npm run preview
```

## 🛠️ التقنيات المستخدمة

### Frontend Framework
- **React 18+** - مكتبة JavaScript للواجهات التفاعلية
- **TypeScript** - للكتابة الآمنة والتطوير المتقدم

- **معالجة اللغة الطبيعية** - لفهم المدخلات المعقدة

### أدوات إضافية

## 📖 كيفية الاستخدام

### 1. إنشاء خريطة ذهنية جديدة
1. اكتب موضوع الخريطة في لوحة الذكاء الاصطناعي
2. اختر نوع المدخل (نص، صورة، صوت، فيديو)
3. اضغط على "إرسال" لتوليد الخريطة تلقائياً

### 2. تحرير وتطوير الخريطة
- **نقرة مزدوجة** على أي عقدة لتحريرها
- **زر التوسيع الذكي** لإضافة أفكار جديدة
- **سحب وإفلات** العقد لإعادة ترتيبها

### 3. حفظ ومشاركة
- **تصدير** بصيغ متعددة (JSON, PNG, PDF, SVG)
- **مشاركة** الرابط مع الآخرين
- **حفظ تلقائي** يحفظ عملك باستمرار
